## 1. Network History

### Timeline of Development

| Period | Development |
|---|---|
| 1960s | ARPANET developed by U.S. Department of Defense (ARPA) — first packet-switching network |
| 1969 | ARPANET connects 4 universities (UCLA, Stanford, UCSB, University of Utah) |
| 1970s | Development of Ethernet by Xerox PARC (Robert Metcalfe) |
| 1974 | TCP/IP protocol proposed by Vint Cerf and Bob Kahn |
| 1983 | ARPANET adopts TCP/IP; considered the birth of the modern Internet |
| 1989 | Tim Berners-Lee invents the World Wide Web (WWW) at CERN |
| 1990s | Commercialization of the Internet; rise of ISPs |
| 2000s onward | Broadband, wireless networks, mobile networks, cloud-based networking |

**Exam Point:** ARPANET is always asked as "the first network" — remember it was funded by ARPA (Advanced Research Projects Agency) for military/research communication.

## 2. Introduction of Network

### Definition
A **computer network** is a collection of two or more interconnected computing devices (computers, servers, printers, etc.) that can exchange data and share resources using a communication medium (wired or wireless).

### Basic Components of a Network
1. **Sender** – device that sends the data
2. **Receiver** – device that receives the data
3. **Medium** – physical/wireless path through which data travels
4. **Message** – the data being transmitted
5. **Protocol** – set of rules governing data communication

### Key Terms
- **Node**: Any device connected to the network (computer, printer, switch, etc.)
- **Link**: The communication path between two nodes
- **Data Communication**: Exchange of data between two devices via a transmission medium

## 3. Uses of Computer Network

| Use Case | Description |
|---|---|
| Resource Sharing | Sharing hardware (printers, scanners) and software across users |
| Information Sharing | Easy access and exchange of files, documents, data |
| Communication | Email, video conferencing, instant messaging |
| Centralized Data Management | Data stored on servers, accessible to authorized users |
| Cost Reduction | Shared resources reduce the need for duplicate hardware/software |
| Remote Access | Access systems/files from a different location |
| E-commerce and Online Services | Online banking, shopping, business transactions |

**Exam Point:** A commonly asked question is "advantages of networking" — cover **resource sharing, communication, cost saving, reliability (backup across systems), scalability**.

---

## 4. Application of Network

### Business Applications
- Resource sharing (files, printers, databases)
- Client-Server model for centralized business operations
- E-commerce (online transactions, digital payments)
- Video conferencing for remote meetings

### Home Applications
- Internet access for information and entertainment
- Person-to-person communication (email, chat, social media)
- Interactive entertainment (online gaming, streaming)
- E-commerce for home users (online shopping)

### Mobile Users
- Access to internet/data over mobile networks
- GPS and location-based services
- Mobile banking and mobile commerce (m-commerce)

### Social Issues
- Networks raise concerns like privacy, data security, and misuse of information — often mentioned briefly as a side-point in application-based questions.

1. **First network** → ARPANET (1969), funded by ARPA.

6. **Uses** → Resource sharing, Communication, Cost reduction, Centralized data, Remote access.
