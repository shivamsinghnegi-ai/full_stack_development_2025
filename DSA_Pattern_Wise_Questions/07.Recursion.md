

### Pattern 1: Single Recursive Call (Linear Reduction)

| #  | Problem Name                                      | LeetCode Link                                                                 | Difficulty | Key Recursive Idea                              | Typical Base Case                     |
|----|---------------------------------------------------|-------------------------------------------------------------------------------|------------|--------------------------------------------------|----------------------------------------|
| 1  | Factorial                                         | (Classic – no direct LC problem, but used everywhere)                        | Easy       | fact(n) = n × fact(n-1)                         | n ≤ 1 → 1                              |
| 2  | Sum of Digits                                     | https://leetcode.com/problems/sum-of-digits-of-a-number/ (variant)           | Easy       | sum(n) = n%10 + sum(n/10)                       | n == 0 → 0                             |
| 3  | Pow(x, n) – recursive (naive version)             | https://leetcode.com/problems/powx-n/                                         | Medium     | power(x,n) = x × power(x, n-1)                  | n == 0 → 1, n < 0 → handle separately |
| 4  | Number of Steps to Reduce a Number to Zero        | https://leetcode.com/problems/number-of-steps-to-reduce-a-number-to-zero/     | Easy       | steps(n) = 1 + steps(n/2 or n-1)                | n == 0 → 0                             |
| 5  | Reverse Integer                                   | https://leetcode.com/problems/reverse-integer/                                | Medium     | build reversed number recursively               | single digit → return it               |
| 6  | Count Sorted Vowel Strings                        | https://leetcode.com/problems/count-sorted-vowel-strings/                     | Medium     | choose next vowel ≥ previous                    | n == 0 → 1                             |
| 7  | Sum of All Subset XOR Totals                      | https://leetcode.com/problems/sum-of-all-subset-xor-totals/                   | Easy       | xor with/without current element                | empty subset → 0                       |
| 8  | Check if Number is a Power of Two                 | https://leetcode.com/problems/power-of-two/                                   | Easy       | n % 2 == 0 && isPower(n/2)                      | n == 1 → true                          |
| 9  | Count Number of 1 Bits                            | https://leetcode.com/problems/number-of-1-bits/                               | Easy       | count(n) = (n&1) + count(n>>1)                  | n == 0 → 0                             |
|10  | Convert Binary Number in a Linked List to Integer | https://leetcode.com/problems/convert-binary-number-in-a-linked-list-to-integer/ | Easy     | value = head.val × 2^len + recurse(rest)        | !head → 0                              |

### Pattern 2: Multiple Recursive Calls (Branching / Tree-style)

| #  | Problem Name                                      | LeetCode Link                                                                 | Difficulty | Key Recursive Idea                              | Base Case                              |
|----|---------------------------------------------------|-------------------------------------------------------------------------------|------------|--------------------------------------------------|----------------------------------------|
| 1  | Fibonacci Number                                  | https://leetcode.com/problems/fibonacci-number/                               | Easy       | fib(n) = fib(n-1) + fib(n-2)                    | fib(0)=0, fib(1)=1                     |
| 2  | Climbing Stairs                                   | https://leetcode.com/problems/climbing-stairs/                                | Easy       | ways(n) = ways(n-1) + ways(n-2)                 | n <= 0 → 0, n==1 → 1                   |
| 3  | N-th Tribonacci Number                            | https://leetcode.com/problems/n-th-tribonacci-number/                         | Easy       | trib(n) = trib(n-1)+trib(n-2)+trib(n-3)        | trib(0)=0, trib(1)=1, trib(2)=1        |
| 4  | Min Cost Climbing Stairs                          | https://leetcode.com/problems/min-cost-climbing-stairs/                       | Easy       | min(cost(n-1), cost(n-2)) + current cost        | n <= 1 → 0                             |
| 5  | Generate Parentheses                              | https://leetcode.com/problems/generate-parentheses/                           | Medium     | add ( or ) if count allows                      | length == 2n → add to result           |
| 6  | Subsets                                           | https://leetcode.com/problems/subsets/                                        | Medium     | include / exclude current element               | end of array → add current subset      |
| 7  | Letter Combinations of a Phone Number             | https://leetcode.com/problems/letter-combinations-of-a-phone-number/          | Medium     | for each digit → try all letters                | no more digits → add combination       |
| 8  | Unique Binary Search Trees (count)                | https://leetcode.com/problems/unique-binary-search-trees/                     | Medium     | for each root → left × right                    | n == 0 or 1 → 1                        |
| 9  | Count Good Numbers                                | https://leetcode.com/problems/count-good-numbers/                             | Medium     | even positions: 5 choices, odd: 4 choices       | n == 0 → 1                             |
|10  | Subsets II (with duplicates)                      | https://leetcode.com/problems/subsets-ii/                                     | Medium     | skip duplicates after sorting                   | end → add subset                       |

### Quick Practice Path Recommendation
1. Start here (very easy & builds confidence):  
   → Climbing Stairs → Fibonacci Number → Number of Steps to Reduce to Zero → Sum of Digits

2. Then try branching:  
   → Subsets → Letter Combinations of a Phone Number → Generate Parentheses

3. Then divide & conquer:  
   → Pow(x, n) → Binary Search (recursive) → Search in Rotated Sorted Array
