# Unix/Linux Commands

### File Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `ls` | `ls` or `ls -l` | List files in directory |
| `cat` | `cat filename` | Display file content |
| `touch` | `touch filename` | Create empty file |
| `cp` | `cp source dest` | Copy file |
| `mv` | `mv source dest` | Move or rename file |
| `rm` | `rm filename` | Delete file |
| `more` | `more filename` | View file page by page |
| `head` | `head filename` | Show first 10 lines |
| `tail` | `tail filename` | Show last 10 lines |

### Directory Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `pwd` | `pwd` | Print current directory path |
| `cd` | `cd dirname` | Change directory |
| `mkdir` | `mkdir dirname` | Create new directory |
| `rmdir` | `rmdir dirname` | Remove empty directory |
| `ls` | `ls -la` | List all files with details |

### Permission Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `chmod` | `chmod 755 file` | Change file permissions |
| `chown` | `chown user file` | Change file owner |

#### Understanding chmod Permissions:
```
r = read    = 4
w = write   = 2
x = execute = 1

chmod 755 → Owner: rwx (7), Group: r-x (5), Others: r-x (5)
chmod 644 → Owner: rw- (6), Group: r-- (4), Others: r-- (4)
```

### Process Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `ps` | `ps` | Show running processes |
| `kill` | `kill PID` | Kill a process by ID |
| `top` | `top` | Show real-time processes |
| `bg` | `bg` | Run process in background |
| `fg` | `fg` | Bring process to foreground |

### Other Important Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `man` | `man command` | Show manual/help of command |
| `grep` | `grep "word" file` | Search text in file |
| `echo` | `echo "Hello"` | Print text to screen |
| `who` | `who` | Show logged-in users |
| `date` | `date` | Show current date & time |
| `clear` | `clear` | Clear terminal screen |
| `whoami` | `whoami` | Show current username |
| `uname` | `uname -a` | Show system information |
| `find` | `find / -name file` | Find a file in system |
| `wc` | `wc filename` | Count lines, words, chars |
