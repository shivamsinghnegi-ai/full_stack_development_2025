# Unix/Linux Commands

### File Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `ls` | `ls` / `ls -l` | List files in current directory |
| `cat` | `cat filename` | Display contents of a file |
| `touch` | `touch filename` | Create a new empty file |
| `cp` | `cp source destination` | Copy a file |
| `mv` | `mv source destination` | Move or rename a file |
| `rm` | `rm filename` | Delete a file |
| `more` | `more filename` | View file content page by page |
| `head` | `head filename` | Show first 10 lines of a file |
| `tail` | `tail filename` | Show last 10 lines of a file |
| `wc` | `wc filename` | Count lines, words, and characters |

### Directory Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `pwd` | `pwd` | Print current working directory path |
| `cd` | `cd dirname` | Change to another directory |
| `mkdir` | `mkdir dirname` | Create a new directory |
| `rmdir` | `rmdir dirname` | Remove an empty directory |

### Permission Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `chmod` | `chmod 755 filename` | Change file permissions |
| `chown` | `chown user filename` | Change file owner |

#### File Permissions Explained:
```
chmod 755 file
  7 = rwx → Owner  (read + write + execute = 4+2+1)
  5 = r-x → Group  (read + execute = 4+0+1)
  5 = r-x → Others (read + execute = 4+0+1)

r = read    = 4
w = write   = 2
x = execute = 1
```

### Process Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `ps` | `ps` | Show currently running processes |
| `kill` | `kill PID` | Terminate a process using its ID |
| `top` | `top` | Show real-time process activity |

### General Utility Commands

| Command | Syntax | Purpose |
|---------|--------|---------|
| `man` | `man command` | Show help/manual for a command |
| `grep` | `grep "word" file` | Search for a word inside a file |
| `echo` | `echo "text"` | Print text to the terminal screen |
| `who` | `who` | Show currently logged-in users |
| `date` | `date` | Display current date and time |
| `clear` | `clear` | Clear the terminal screen |
| `whoami` | `whoami` | Show current logged-in username |
| `uname` | `uname -a` | Show system/OS information |
| `find` | `find / -name filename` | Search for a file in the system |
