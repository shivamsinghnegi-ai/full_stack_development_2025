# Features of Unix/Linux

1. **Multi-user** — Multiple users can use the system simultaneously.

2. **Multitasking** — Multiple processes can run at the same time.

3. **Portability** — Written in C, so it can run on different hardware platforms.

4. **Security** — File permissions (read/write/execute), user authentication, encryption.

5. **Open Source (Linux)** — Source code is freely available; anyone can modify and distribute.

6. **Hierarchical File System** — Files are organized in a tree structure starting from root `/`.

7. **Shell Scripting** — Supports automation through shell scripts (batch commands).

8. **Networking Support** — Built-in support for TCP/IP, FTP, SSH, and other protocols.

9. **Stability & Reliability** — Rarely crashes; can run for years without restart.

10. **Case Sensitive** — `File.txt` and `file.txt` are treated as different files.
