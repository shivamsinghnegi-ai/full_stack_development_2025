# Complete Jenkins Setup & Execution Guide (Windows & macOS)

## Overview

This guide explains how to install prerequisites, run Jenkins, configure Java/Maven/Node.js, create jobs, and execute commands on both **Windows** and **macOS**.

---

# 1. What is Jenkins?

Jenkins is an automation server used for:

* Continuous Integration (CI)
* Continuous Delivery/Deployment (CD)
* Running builds automatically
* Testing code
* Deploying applications
* Scheduled tasks

---

# 2. Prerequisites

Jenkins requires:

* **Java 21+** (modern Jenkins versions)
* Optional build tools:

  * Maven (Java projects)
  * Node.js + npm (JavaScript projects)
  * Git

---

# 3. Install Java 21

## macOS (Homebrew)

```bash
brew install openjdk@21
sudo ln -sfn /opt/homebrew/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
export JAVA_HOME=/opt/homebrew/opt/openjdk@21/libexec/openjdk.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH
java -version
```

## Windows

1. Download JDK 21 from Temurin / Oracle.
2. Install it.
3. Set environment variables:

   * `JAVA_HOME=C:\Program Files\Java\jdk-21`
   * Add `%JAVA_HOME%\\bin` to PATH

Check:

```cmd
java -version
```

---

# 4. Install Maven

## macOS

```bash
brew install maven
mvn -version
```

Recommended Maven Home:

```text
/opt/homebrew/opt/maven/libexec
```

## Windows

1. Download Maven zip.
2. Extract to:

```text
C:\apache-maven-3.9.x
```

3. Set:

* `MAVEN_HOME=C:\apache-maven-3.9.x`
* Add `%MAVEN_HOME%\\bin` to PATH

Check:

```cmd
mvn -version
```

# 6. Run Jenkins WAR File

## Download Jenkins WAR

Download `jenkins.war` from Jenkins official website.

---

## macOS

If file is in Downloads:

```bash
cd ~/Downloads
java -jar jenkins.war
```

If port 8080 busy:

```bash
java -jar jenkins.war --httpPort=9090
```

## Windows

If file is in Downloads:

```cmd
cd %USERPROFILE%\Downloads
java -jar jenkins.war
```

Different port:

```cmd
java -jar jenkins.war --httpPort=9090
```

# 8. Initial Admin Password

## macOS

```bash
cat ~/.jenkins/secrets/initialAdminPassword
```

## Windows

```cmd
type %USERPROFILE%\.jenkins\secrets\initialAdminPassword
```

If `%USERPROFILE%` does not work on some laptops, use any of these alternatives:

```cmd
echo %HOMEPATH%
echo %USERPROFILE%
cd %HOMEDRIVE%%HOMEPATH%
type C:\Users\%USERNAME%\.jenkins\secrets\initialAdminPassword
```

You can also manually open:

```text
C:\Users\YourUsername\.jenkins\secrets\initialAdminPassword
```

Paste password into browser.

# 17. Run Jenkins in Background

## macOS

```bash
nohup java -jar ~/Downloads/jenkins.war > jenkins.log 2>&1 &
```

## Windows

Use NSSM or run Jenkins as Windows service.

---

# 18. Stop Jenkins

## macOS

Press `Ctrl + C` in terminal running Jenkins.

## Windows

Close CMD window or stop service.

---

# 19. Common Errors

## Error: Unable to access jarfile jenkins.war

Go to correct folder:

```bash
cd ~/Downloads
```

## Java version too old

Install Java 21.

## Port 8080 already in use

Use:

```bash
java -jar jenkins.war --httpPort=9090
```
