# Overview of Modern Java Web Technologies
### Evolution Timeline

```
1995 - Java Born
1997 - Servlets Introduced
1999 - JSP (JavaServer Pages)
2001 - J2EE (Java 2 Platform, Enterprise Edition)
2006 - Java EE 5 (Renamed from J2EE)
2017 - Java EE to Jakarta EE transition
2018+ - Spring Framework dominance
```

### Core Technologies

#### 1. **Servlets**
- Java classes that handle HTTP requests
- Run on web servers
- Foundation for web applications

```java
@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, 
                         HttpServletResponse response) throws IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello from Servlet!</h1>");
    }
}
```

#### 2. **JSP (JavaServer Pages)**
- HTML with embedded Java code
- Easier than Servlets for UI
- Compiled into Servlets

```jsp
<%@ page language="java" contentType="text/html" %>
<html>
<body>
    <h1>Welcome <%= request.getParameter("name") %></h1>
    <p>Current time: <%= new java.util.Date() %></p>
</body>
</html>
```

#### 3. **JSTL (JSP Standard Tag Library)**
- Simplifies JSP development
- Tag-based approach

```jsp
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<c:forEach items="${products}" var="product">
    <p>${product.name} - ${product.price}</p>
</c:forEach>
```

#### 4. **JavaBeans**
- Reusable software components
- Follow specific conventions
- Used for data encapsulation

```java
public class Student {
    private String name;
    private int rollNumber;
    
    // Getter
    public String getName() { return name; }
    
    // Setter
    public void setName(String name) { this.name = name; }
    
    // Constructor
    public Student() {}
}
```

### Modern Frameworks

#### **Spring Framework**
- Most popular Java framework
- Dependency Injection
- Aspect-Oriented Programming
- Transaction Management

#### **Spring Boot**
- Simplified Spring development
- Auto-configuration
- Embedded servers
- Production-ready features

#### **Hibernate (ORM)**
- Object-Relational Mapping
- Simplifies database operations
- Eliminates repetitive JDBC code
