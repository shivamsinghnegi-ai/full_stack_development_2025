# Java Servlet with Apache Tomcat using IntelliJ IDEA (Community Edition)


## 1️. Install Prerequisites

### Install JDK
- Install **JDK 11 or 17**
- Verify installation:
```bash
java -version
```

### Download Apache Tomcat 9
- Download from: https://tomcat.apache.org/download-90.cgi
- Extract to:
```text
C:\apache-tomcat-9.0.xxx
```

---

## 2️. Create Java Project in IntelliJ IDEA

1. Open **IntelliJ IDEA**
2. Click **File → New Project**
3. Select **Java**
4. Choose installed JDK
5. Project name:
```text
servlet-demo
```
6. Click **Create**

---

## 3️. Create Required Folder Structure

```text
servlet-demo
 ├── src
 │    └── HelloServlet.java
 └── web
      └── WEB-INF
           └── web.xml
```

- `src` → Sources Root  
- `web` → Web Resource Root

---

## 4️. Add Servlet API Library

1. Go to:
```text
C:\apache-tomcat-9.0.xxx\lib\servlet-api.jar
```

2. IntelliJ:
- File → Project Structure
- Libraries → + → Java
- Select `servlet-api.jar`
- Apply → OK

---

## 5️. Create First Servlet

```java
import java.io.IOException;
import javax.servlet.ServletException;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

public class HelloServlet extends HttpServlet {

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {

        resp.setContentType("text/plain");
        resp.getWriter().println("Servlet is running successfully!");
    }
}

```

---

## 6️. Configure web.xml

```xml
<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee
         http://xmlns.jcp.org/xml/ns/javaee/web-app_4_0.xsd"
         version="4.0">

    <servlet>
        <servlet-name>HelloServlet</servlet-name>
        <servlet-class>HelloServlet</servlet-class>
    </servlet>

    <servlet-mapping>
        <servlet-name>HelloServlet</servlet-name>
        <url-pattern>/hello</url-pattern>
    </servlet-mapping>

</web-app>

```

---

## 7️. Add Web Facet

File → Project Structure → Modules → Facets → + → Web  
Set web root: `web`  
Descriptor: `web/WEB-INF/web.xml`

---

## 8️. Create WAR Exploded Artifact

File → Project Structure → Artifacts  
+ → Web Application: Exploded

---

## 9️. Build Artifact

Build → Build Artifacts → servlet:war exploded

---

## 10. Configure Tomcat

Run → Edit Configurations → Tomcat Server → Local  
Deployment:
- Add artifact
- Context: `/servlet_war_exploded`

---

## 11. Run Server

Click  Run  
Wait for successful deployment message.

---

## 12. Open in Browser

```text
http://localhost:8080/servlet_war_exploded/hello
```
