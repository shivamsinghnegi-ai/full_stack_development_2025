## Semiconductor Memories
 
### 1. What Are Semiconductor Memories?
 
Semiconductor memories are electronic memory chips made using semiconductor (silicon) technology. They are used for **registers, cache, and main memory (RAM/ROM).**
 
---
 
### 2. Classification of Semiconductor Memories
 
```
Semiconductor Memory
├── Volatile (loses data on power off)
│   └── RAM (Random Access Memory)
│       ├── SRAM (Static RAM)
│       └── DRAM (Dynamic RAM)
│
└── Non-Volatile (retains data on power off)
    └── ROM (Read Only Memory)
        ├── MROM
        ├── PROM
        ├── EPROM
        └── EEPROM / Flash
```
 
---
 
### 3. Key Properties of Semiconductor Memories
 
| Property | Explanation |
|---|---|
| **Random Access** | Any memory location can be accessed in the same time, regardless of address |
| **Volatile** (for RAM) | Data is lost when power is removed |
| **Non-volatile** (for ROM) | Data is preserved without power |
| **Read/Write** (for RAM) | Data can be read and written freely |
| **Read Only** (for ROM) | Data can only be read (in basic types) |
