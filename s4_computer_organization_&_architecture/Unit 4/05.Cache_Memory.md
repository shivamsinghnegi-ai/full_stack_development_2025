# Cache Memory
 
### 1. What is Cache Memory?
 
Cache is a **small, very fast** memory placed **between the CPU and main memory (RAM).**
 
Its job: **store frequently used data** so the CPU does not have to go to slow main memory every time.
 
```
CPU  ←──fast──→  Cache  ←──slow──→  Main Memory (RAM)
                (SRAM)                   (DRAM)
```
 
- Cache uses **SRAM** (fast, expensive)
- Main memory uses **DRAM** (slow, cheap)
- Cache is **much smaller** than main memory — but much faster
 
---
 
### 2. Why is Cache Needed?
 
**The Problem — CPU-Memory Speed Gap:**
 
```
CPU speed      ~  1 ns
Main Memory    ~  100 ns
Speed gap      =  100x
```
 
Without cache, CPU would waste ~99 cycles waiting for data from RAM.
 
**The Solution — Cache:**
Keep frequently used data in cache. If CPU finds data in cache (HIT), it gets it in ~2–5 ns. Only if data is not there (MISS) does it go to RAM.
 
---
 
### 3. Cache Levels — L1, L2, L3
 
| Level | Location | Size | Speed | Shared? |
|---|---|---|---|---|
| **L1** | Inside CPU core | 32–64 KB per core | Fastest (~2–5 ns) | No — per core |
| **L2** | Inside/near CPU core | 256 KB – 1 MB per core | Fast (~5–10 ns) | Usually per core |
| **L3** | On CPU chip (outside cores) | 4–32 MB | Moderate (~10–30 ns) | Yes — shared across all cores |
 
**Access order:** CPU first checks L1 → then L2 → then L3 → then RAM.
 
```
CPU Core
  ├── L1 Cache  (fastest, smallest — first check)
  ├── L2 Cache  (second check)
  └── L3 Cache  (third check — shared)
        └── Main Memory (RAM) — last resort
```
 
---
 
### 4. Hit and Miss 
 
**HIT:** CPU requests data → data **found in cache** → fast access 
 
**MISS:** CPU requests data → data **NOT in cache** → must fetch from RAM 
 
**Hit Rate (h):** Fraction of memory accesses that result in a cache hit.
```
Hit Rate = (Number of Hits) / (Total Memory Accesses)
```
 
**Miss Rate:**
```
Miss Rate = 1 − Hit Rate
```
 
**Example:**
- 1000 memory accesses, 950 found in cache
- Hit Rate = 950/1000 = **0.95 (95%)**
- Miss Rate = 1 − 0.95 = **0.05 (5%)**
