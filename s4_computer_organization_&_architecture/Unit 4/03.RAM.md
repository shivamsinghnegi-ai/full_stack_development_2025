## RAM — Random Access Memory
 
### 1. What is RAM?
 
RAM is the **main memory** of a computer. It temporarily stores data and instructions that the CPU is currently using.
 
- It is **volatile** — all data is lost when power is cut.
- The CPU can **read from and write to** RAM freely.
- All currently running programs and their data reside in RAM.
 
---
 
### 2. Types of RAM
 
RAM is divided into two main types:
 
| | SRAM | DRAM |
|---|---|---|
| Full Name | Static RAM | Dynamic RAM |
| Storage Element | Flip-flop (6 transistors) | Capacitor + Transistor |
| Refresh Needed? | No | Yes (periodic refresh) |
| Speed | Very fast | Slower than SRAM |
| Cost | Very expensive | Less expensive |
| Power Consumption | Higher | Lower |
| Density (cells/area) | Low | High |
| Size | Smaller capacity | Larger capacity |
| Used In | Cache memory (L1, L2, L3) | Main memory (RAM modules) |
 
---
 
### 3. SRAM — Static RAM
 
#### How SRAM Works
 
SRAM stores each bit using a **flip-flop circuit** made of **6 transistors**.
 
- A flip-flop is a bistable circuit — it holds either a 0 or a 1 **as long as power is supplied.**
- It does **not need to be refreshed** because transistors actively maintain the state.
- The bit "locks" itself into one of two stable states.
 
```
     VCC
      |
   ┌──┴──┐   ┌──────┐
   │ T1  │───│  T3  │───┐
   └──┬──┘   └──────┘   │
      │                  │
   ┌──┴──┐   ┌──────┐   │
   │ T2  │───│  T4  │   │
   └──┬──┘   └──────┘   │
      │                  │
     GND    (simplified 6-T flip-flop)
```
 
*(Simplified view — actual circuit uses 4 transistors for the latch + 2 access transistors)*
 
#### Key Characteristics of SRAM
 
- **Fast:** Access time of 2–10 ns
- **Expensive:** 6 transistors per bit → more silicon area → more cost
- **No refresh needed:** Simpler controller logic
- **Used in:** L1, L2, L3 cache memory inside/near the CPU
 
---
 
### 4. DRAM — Dynamic RAM
 
#### How DRAM Works
 
DRAM stores each bit using **one capacitor and one transistor**.
 
- A capacitor stores charge: **charged = 1, discharged = 0.**
- The transistor acts as a switch to read/write the capacitor.
- **Problem:** Capacitors **leak charge** over time (within milliseconds).
- **Solution:** The memory controller must **refresh** (re-charge) every cell periodically — typically every **64 ms** — to prevent data loss.
 
```
        ┌─────────────────┐
        │   Word Line     │ (selects row)
        │        │        │
        │   ┌────┴───┐    │
        │   │Transistor│  │
        │   └────┬───┘    │
        │        │        │
        │   ┌────┴───┐    │
        │   │Capacitor│   │  ← Stores 1 bit
        │   └────────┘    │
        │                 │
        │   Bit Line ─────┤ (data in/out)
        └─────────────────┘
```
 
#### Why is Refresh Needed?
 
| Event | What Happens |
|---|---|
| Write '1' | Capacitor is charged |
| After ~10–100 ms | Capacitor leaks charge → bit becomes '0' (data lost!) |
| Refresh | Controller reads each row and rewrites it before charge is lost |
 
The word "**Dynamic**" in DRAM refers to this **dynamic (changing) charge** that must be constantly refreshed.
 
#### Key Characteristics of DRAM
 
- **Slower:** Access time ~50–100 ns
- **Cheap:** Only 1 transistor + 1 capacitor per bit → high density, low cost
- **Needs refresh:** Adds complexity to the memory controller
- **Used in:** Main memory (DDR4/DDR5 RAM modules in your PC/laptop)
 
---
 
### 5. SRAM vs DRAM — Complete Comparison 
 
| Feature | SRAM | DRAM |
|---|---|---|
| Storage element | 6-transistor flip-flop | 1 capacitor + 1 transistor |
| Data retention | As long as power is on | Needs periodic refresh |
| Refresh required | No | Yes (every ~64 ms) |
| Access speed | Very fast (2–10 ns) | Slower (50–100 ns) |
| Cost per bit | High | Low |
| Power usage | High (transistors always on) | Low (capacitor-based) |
| Density | Low (more area per bit) | High (smaller cell) |
| Capacity | Small (KBs to MBs) | Large (GBs) |
| Primary use | Cache memory | Main memory |
| Complexity | Simple (no refresh logic) | Complex (needs refresh circuit) |
 
---
 
### 6. DDR RAM (Just for Awareness)
 
Modern DRAM comes in **DDR (Double Data Rate)** versions:
 
| Type | Full Name | Notes |
|---|---|---|
| DDR3 | Double Data Rate 3 | Older laptops/desktops |
| DDR4 | Double Data Rate 4 | Current standard |
| DDR5 | Double Data Rate 5 | Latest, fastest |
 
"Double Data Rate" means data is transferred on **both the rising and falling edges** of the clock signal, doubling throughput compared to single data rate.
