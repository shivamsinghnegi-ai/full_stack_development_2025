# Cache Replacement Policies
 
When cache is **full** and a new block must be loaded, which existing block gets replaced?
 
| Policy | Rule | Notes |
|---|---|---|
| **LRU** (Least Recently Used) | Replace block used **longest ago** | Most commonly used, good performance |
| **FIFO** (First In First Out) | Replace block that came in **first** | Simple, less effective than LRU |
| **LFU** (Least Frequently Used) | Replace block used **fewest times** | Complex to implement |
| **Random** | Replace a **random** block | Simple, unpredictable |
 
> **Exam note:** LRU is the most commonly asked replacement policy.

## Summary 
 
### ROM Types — Quick Reference
 
| Type | Erase Method | Reprogrammable? |
|---|---|---|
| MROM | Cannot erase | No |
| PROM | Cannot erase | No (OTP) |
| EPROM | UV Light | Yes (out-of-circuit) |
| EEPROM | Electrical (byte) | Yes (in-circuit) |
| Flash | Electrical (block) | Yes (in-circuit) |
 
### Cache — Quick Reference
 
- **Why cache?** Bridge speed gap between CPU (1 ns) and RAM (100 ns)
- **Levels:** L1 → L2 → L3 → RAM (in access order)
- **Mapping:** Direct < Set-Associative < Fully Associative (flexibility)
- **Write-Through:** Every write → RAM immediately
- **Write-Back:** Write to RAM only on eviction (dirty bit)
- **Replacement:** LRU is most common
 
### AMAT Formula
 
```
AMAT = Hit Time + (Miss Rate × Miss Penalty)
Miss Rate = 1 − Hit Rate
```

## Numerical Practice Set
 
Solve these before the exam:
 
**Q1.** Cache access time = 15 ns, Main memory time = 150 ns, Hit rate = 0.92. Find AMAT.
 
> **Answer:** Miss rate = 0.08 → AMAT = 15 + (0.08 × 150) = 15 + 12 = **27 ns**
 
---
 
**Q2.** In 8000 memory references, 7200 were hits. Cache time = 8 ns, MM time = 120 ns. Find AMAT.
 
> **Answer:** Hit rate = 7200/8000 = 0.9 → Miss rate = 0.1 → AMAT = 8 + (0.1 × 120) = 8 + 12 = **20 ns**
 
---
 
**Q3.** 4-way set-associative cache. Cache has 8 sets. Main memory block 29 maps to which set?
 
> **Answer:** Set = 29 mod 8 = **5**
 
---
 
**Q4.** Two-level cache: L1 time = 5 ns, L1 hit rate = 0.90. L2 time = 25 ns, L2 hit rate = 0.85. MM time = 200 ns. Find AMAT.
 
> **Answer:**
> AMAT = 5 + 0.10 × (25 + 0.15 × 200)
> = 5 + 0.10 × (25 + 30)
> = 5 + 0.10 × 55
> = 5 + 5.5
> = **10.5 ns**
